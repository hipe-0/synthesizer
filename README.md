DIY synthesizer, inspired by the Moog model D synthesizer

Features
- 26 faders
- 3 waveforms
- 1 low pass filter
- Core : teensy 4.1 microcontroller board
- LED display
- LCD display
- usb midi In
- usb host (to plug a usb midi keyboard on it)
